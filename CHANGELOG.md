# Orbit v1.0.1

**平台：** Windows · Android

## 新功能与改进

### 课表

- 支持左右滑动切换周次
- 修复周切换时页面闪烁
- 有课表但当前周无课程时，仍可翻页浏览其他周

### 界面

- 课程详情底部按钮窄屏自适应（编辑 / 备注 / 删除）
- 课表星期选择器垂直对齐优化
- 「接下来」列表状态竖条对齐优化

### Windows

- 单实例启动：重复打开时激活已有窗口，不新增任务栏图标
- 托盘「退出」加速至约 1 秒内
- 应用显示名称统一为 **Orbit**

## 修复

- 修复设置中「清除所有课表」后，课表页未回到「尚未导入课表」空态的问题
- 修复课表翻页时的短暂闪烁
