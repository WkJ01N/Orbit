# Orbit v1.1.1

**平台：** Windows · Android

## 新功能与改进

### Android

- 课前 / 打卡提醒通过 `AlarmManager.oneShotAt` 逐条注册，App 退到后台或被系统杀进程后仍可触发
- 每 6 小时维护闹钟，设备重启后自动恢复排程
- 设置页新增「后台提醒（Android）」区块：电池优化开关、OriginOS / iQOO 自启动提示、「测试后台提醒（1 分钟）」按钮

### 课表

- 宽屏表头改用 `SliverPersistentHeader` 吸顶，与表体共享同一垂直滚动 viewport；导入多时段课表出现纵向滚动条时，周日期与下方课程列仍对齐

## 修复

- 修复宽屏导入课表后纵向滚动条占用表体宽度、导致表头与课程列水平错位的问题

## 其他

- 版本号 1.1.1（`pubspec.yaml` build `+6`）
- 测试增至 76 项（含宽屏表头 / 表体对齐、AlarmManager 注册表用例）

# Orbit v1.1.0

**平台：** Windows · Android

## 新功能与改进

### 课表

- 宽屏模式下表头与课程列对齐（统一横向滚动，消除滚动条占位错位）
- 左侧时间线各时段固定行高，不再因单日课程变化而拉伸
- 当前时间红线仅在课表有效时段内显示，并按真实时间比例定位；**且仅在今天对应列显示**（窄屏需选中今天，宽屏不横跨整周）
- 已导入课表后，无课周不再显示「立即导入」按钮；副标题改为引导切换周次
- 课表 AppBar 窄屏布局优化：换周按钮与搜索 / 今天按钮不再重叠
- 课表 AppBar 统一为标准 AppBar，与其他页面顶栏样式一致
- 横向滚动检测边缘状态时不再触发整表 rebuild，宽屏横滚更顺滑
- 课程 Chip 高亮与当前时间红线共用分钟级时间源，每分钟同步更新
- 时间红线、删除菜单改用主题色；课程 Chip 字号继承 `textTheme`，支持系统字号缩放
- 课表加载态改为骨架屏占位，减少白屏跳变
- 课表单元格增加重绘隔离（`RepaintBoundary`），降低时间线更新时的绘制开销

### 接下来

- 「即将开始」与倒计时分两行显示，避免窄列内换行错乱
- 「上课中」状态徽章垂直居中对齐
- 列表扁平化懒加载，去除 `IntrinsicHeight`，长列表滚动更流畅
- 加载态改为骨架屏占位

### 课程

- 手动添加 / 编辑课程时可填写学院名称；详情页学院为空时不显示该行
- 手动课程未填编号时使用唯一内部编号，避免同日同时间冲突覆盖

### 界面与导航

- 底部 Tab 切换增加淡入过渡（保留 IndexedStack 页面状态）
- 宽屏（≥600px）课程详情、编辑、备注改为居中对话框；窄屏保持底部 Sheet
- 课表与「接下来」FAB 仅在数据加载完成后显示，行为一致

### 提醒

- 保存课程后显示「已排定 N 条提醒」；系统未写入 pending 时明确提示检查精确闹钟、电池优化与自启动（OriginOS / iQOO）
- 精确闹钟调度失败时自动降级为非精确模式，降低完全无提醒的概率
- 课前 / 打卡提醒优先使用 Android `alarmClock` 模式，并保留精确 / 非精确降级链
- 2 小时内到期的课前 / 打卡提醒额外注册进程内 Timer，App 存活时直接弹出通知作为兜底
- 使用设备 IANA 时区（`flutter_timezone`）配置提醒调度，减少时区映射偏差
- 「重新同步提醒」在仅开启签到 / 次日摘要时也可使用
- 手动添加课程默认开始时间为下一整点或半点，减少因时间已过去而静默跳过提醒

### Android

- 系统闹钟增加 `SET_ALARM` 包可见性声明，并支持 OriginOS / Vivo 时钟 Intent 回退

## 稳定性

- 修复冷启动时提醒排程未使用已保存设置的问题
- 排程完成后校验系统 pending 数量，捕获 OEM 静默丢弃精确闹钟的情况

## 修复

- 修复当前时间红线在非今天列 / 非选中天仍显示的问题
- 修复课程详情弹窗删除时报错「Cannot use ref after the widget was disposed」
- 修复手动添加课程无法填写学院信息
- 修复手动课程提醒已排入 pending 但到点不触发（时区、Android 投递链与近端 Timer 兜底）
- 修复课前 / 打卡提醒对手动添加与导入课程不生效（含排程验证与权限相关改进）
- 修复课程详情页编辑、备注按钮点击无效（删除见上）
- 修复 Android 系统闹钟「无法打开系统闹钟」
- 修复 PC 宽屏手动添加课程后表头与列不对齐

## 其他

- 版本号 1.1.0（`pubspec.yaml` build `+5`）
- 测试增至 70 项（含红线当天列、提醒时区 / 近端兜底、倒计时与 Chip 状态用例）

# Orbit v1.0.3

**平台：** Windows · Android

## 新功能与改进

### 导入与课表

- 导入时若与本地存在重复周，可选择「整周替换」或「合并并覆盖冲突」
- 手动添加 / 编辑课程时，时间冲突自动覆盖旧课，并提示覆盖数量
- 设置中新增默认周次：智能 / 本周 / 最早有课周（默认智能）
- 课程搜索支持简体与繁体中文互搜；搜索页增加加载态、错误提示与结果截断说明
- 当周无课时提供「立即导入」入口

### 提醒

- 课表增删改后若提醒重排部分失败，SnackBar 提示失败数量
- 提醒重排部分失败时在全局（设置页除外）显示 MaterialBanner，便于重新同步
- Android 权限引导在未允许精确闹钟时跳转精确闹钟授权页

### Windows

- 修复长时间锁屏或合盖后托盘图标偶发消失（升级 tray_manager，唤醒 / 聚焦时自检恢复）
- 点击系统通知时自动唤出主窗口
- 首次关闭到托盘前先显示提示 SnackBar
- 托盘菜单与 tooltip 随语言切换更新

### 设置与界面

- 导出 / 备份 / 还原操作增加进行中状态，防止重复点击
- 课程编辑与备注保存按钮增加防双击
- 清除课表、批量删除、开机启动等失败时使用语义化错误文案

## 稳定性

- 提醒重排部分失败时写入全局状态，避免静默丢失提醒而无感知

## 修复

- 修复从搜索结果打开课程详情可能失败
- 修复 Android 导出取消仍显示成功
- 修复 Android 精确闹钟权限跳转不准确

## 其他

- 版本号升至 1.0.3（`pubspec.yaml` build `+4`）
- 测试增至 54 项（含简繁搜索用例）

# Orbit v1.0.2

**平台：** Windows · Android

## 新功能与改进

### 导入与导出

- 支持将课表导出为 JSON 或 xlsx，便于备份与迁移
- 支持从 JSON 备份恢复课表
- 导入页增加 xlsx 格式说明；解析失败时显示本地化错误提示

### 课表

- 支持手动添加与编辑单节课程（课表网格与「接下来」页 FAB）
- 新增课程搜索，可按科目、课室、教师等关键词查找

### 提醒

- 点击课前通知可跳转至对应课程详情
- Android 电池优化豁免改为可双向开关，并与系统真实状态同步
- Android 提醒相关权限分项展示授权状态，可跳转系统设置
- 启用 `RebootBroadcastReceiver`，设备重启后自动维护闹钟

### 设置与界面

- 提醒开关操作失败时显示 SnackBar 反馈
- 提醒重排失败时在设置页显示 MaterialBanner 提示

## 稳定性

- 提醒重排（`rescheduleAll`）串行化，避免并发竞态
- 后台维护回调增加 try/catch 与调试日志
- 冷启动时通知 payload 延后至设置初始化完成后再处理
- 应用回到前台时以 6 小时 debounce 重新同步提醒
- 时区配置统一至 `timezone_utils`（UTC 偏移，适配澳门 / 中国大陆）

## 修复

- 修复 Android 电池优化豁免开启后无法关闭的问题
- 语言切换后自动 resync 提醒与界面
- 改进删除、导入等操作的错误处理

## 其他

- 版本号升至 1.0.2（`pubspec.yaml` build `+3`）
- 测试增至 51 项（含备份往返与时区配置用例）
